# UCD Generative AI Project - sCurVy: Intelligent Resume & Cover Letter Builder

This notebook-captures an OpenAI API key, mounts Drive to pull a curated `ResumeTips.txt` knowledge base, chunks and embeds those tips with SentenceTransformer `all-MiniLM-L6-v2`, builds a FAISS index for similarity search, retrieves and filters advice against the applicant’s uploaded resume, and then orchestrates the OpenAI Assistants API end-to-end: it creates the assistant through `openai.beta.assistants.create` with `file_search` and `code_interpreter` tools plus a custom NewsAPI-powered `fetch_news_summary` function (implemented with `requests` and an inline `NEWS_API_KEY`), manages threads and runs while responding to tool-calls by looping on `openai.beta.threads.runs.retrieve` until completion, injects the RAG-derived context and job-specific prompts before messaging, and finally surfaces the assistant’s responses in Markdown so the workflow can iteratively craft tailored resume edits and cover-letter drafts grounded in both retrieved tips and fresh company news.
